# React Todo App

A minimalistic, customisable todo app.

## Features

- CRUD operations on the task
- Tasks saved locally
- Edit task.

## Built with

- React Js, Css
- React Hooks : useState, useEffect, useRef, useContext
- have used context for the state management as the task is not my bigger, but i can use redux as well.
